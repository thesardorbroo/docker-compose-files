# Docker Compose Configurations

This repository contains ready-to-use **Docker Compose configurations** for all services used in the project.

## Purpose

The main goal of this repository is to provide a centralized place where all service configurations are stored and maintained. Using these configurations, developers can quickly start required services in a local or development environment without manual setup.

## What is included

- Preconfigured **docker-compose.yml** files
- Configurations for all project services
- Network and environment setup for service interaction
- Ready-to-run containers for development and testing
